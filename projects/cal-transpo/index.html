Cal Transpo Operations Learning Materials

This project organizes Python-based learning materials for the Cal Transportation Team at UC Berkeley, focused on practical transportation analytics and simulation skills.

  • Python and Data Analytics: Python fundamentals, pandas/NumPy workflows, and data visualization.
  • Real-World Data Analysis: crash, census, and traffic data integration with geospatial processing.
  • Traffic Simulation: simulation basics, scenario experiments, and interpretation of performance metrics.

The materials are designed to help team members move from core coding skills to transportation-focused analysis workflows used in research and operations projects.
