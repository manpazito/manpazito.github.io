Plane Boarding Simulator

A discrete-time, agent-based simulation framework for studying commercial aircraft boarding procedures. This project models passengers, aircraft geometry, overhead bins, and movement constraints to support controlled experiments in boarding efficiency.

Developed for INDENG 174: Simulation for Enterprise-Scale Systems, University of California, Berkeley (Fall 2025).

GitHub Repo

The simulator models aircraft boarding as a discrete-time, agent-based process with passengers, seat assignments, aisle movement constraints, and overhead-bin interactions. It supports controlled comparisons of boarding strategies and reports metrics such as total boarding time, seat timing, and congestion.

For complete setup instructions, CLI usage, configuration options, and analysis outputs, see the GitHub repository linked above.
