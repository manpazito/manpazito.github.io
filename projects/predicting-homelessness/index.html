Predicting Homelessness

This project builds predictive models for homelessness rates across the United States using HUD Point-in-Time (PIT) data from 2011-2017, combined with demographic, economic, housing-market, environmental, and safety-net indicators.

The analysis uses exploratory data analysis and supervised learning to identify drivers of homelessness and compare model outputs with HUD's Market Predictors of Homelessness findings.

  • Models evaluated: Linear Regression, Regression Trees, Random Forest, and Gradient Boosting.
  • Best test performance: Random Forest (R2 = 0.819), followed by Gradient Boosting (R2 = 0.803).
  • Focus: robust prediction quality and practical interpretation for policy-facing decisions.
